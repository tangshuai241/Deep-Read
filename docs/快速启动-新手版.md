# DeepRead 快速启动（新手版）

5 步跑起来，不需要懂 Python。

---

## 第 1 步：安装 Python

1. 打开 https://www.python.org/downloads/
2. 下载 Python 3.9 或更高版本（推荐 3.10+）
3. 运行安装程序，**务必勾选 "Add Python to PATH"**
4. 安装完成后，打开 PowerShell，输入 `python --version`，看到版本号即成功

## 第 2 步：下载项目

把 DeepRead 项目文件夹放到你喜欢的位置，例如：

```
E:\DeepRead\
```

确保文件夹内有这些文件：
- `install.ps1`、`start.ps1`、`start-bot.ps1`
- `cli.py`、`init.py`、`server.py`
- `requirements.txt`、`config.example.yaml`

## 第 3 步：运行安装脚本

在项目目录下，打开 PowerShell，运行：

```powershell
.\install.ps1
```

如果提示"无法加载文件"，先执行：

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

脚本会自动：
1. 检查 Python 版本
2. 创建 `.venv` 虚拟环境
3. 安装所有依赖
4. 引导你完成初始化配置
5. 运行健康检查

## 第 4 步：配置 Obsidian 和 API Key

打开 `config.yaml`，修改以下关键配置：

```yaml
paths:
  vault_dir: "E:\\Documents\\知识库"        # 你的 Obsidian vault 路径
  notes_dir: "E:\\Documents\\知识库\\读书\\笔记"  # 笔记输出目录

llm:
  provider: "deepseek"                      # deepseek / anthropic / openai
  model: "deepseek-v4-pro"
  api_key: "sk-xxxxxxxx"                    # 或设置环境变量 DEEPSEEK_API_KEY
```

### 获取 API Key

| LLM | 获取地址 | 环境变量 |
|-----|---------|---------|
| DeepSeek | https://platform.deepseek.com/api_keys | `DEEPSEEK_API_KEY` |
| Anthropic | https://console.anthropic.com/ | `ANTHROPIC_API_KEY` |
| OpenAI | https://platform.openai.com/api-keys | `OPENAI_API_KEY` |

**推荐**：设置环境变量而不是写在 config.yaml 里，避免泄露。

### 配置飞书 Bot（可选）

飞书 Bot 需要人工完成，无法自动化。步骤：

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用 → 机器人
3. 添加权限：`im:message.p2p_msg:readonly`、`im:message:send_as_bot`
4. 订阅事件：`im.message.receive_v1`
5. 发布应用，获取 App ID / App Secret
6. 在终端运行 `lark-cli auth login` 完成登录

飞书不配置也能用 DeepRead，只是在终端/Web 控制台操作，无法用手机发消息。

## 第 5 步：打开 Web 控制台 + 启动飞书 Bot

```powershell
# 启动 Web 控制台
.\start.ps1

# 另开一个终端，启动飞书 Bot（可选）
.\start-bot.ps1
```

Web 控制台地址：http://127.0.0.1:8765/

## 最低可用路径（不启用 Wiki 也能读书）

如果你不想配置 LLM-Wiki，完全可以只读书：

1. `config.yaml` 中把 `integrations.wiki.enabled` 设为 `false`
2. 飞书 Bot 可选，直接在 Claude Code 或终端对话
3. 不启用 Wiki 只是联想和双链质量会下降，不影响基本精读流程

## 排障入口

| 遇到问题 | 运行这个 |
|---------|---------|
| 整体检查 | `python cli.py doctor` |
| 深度检查 | `python cli.py doctor --deep` |
| 飞书 Bot 状态 | `python cli.py bot status` |
| 笔记质量 | `python cli.py quality "路径/笔记.md"` |
| 飞书 Bot 不工作 | 查看 `logs/feishu_bot.log` |

## 常见问题

### Q: 运行 .ps1 脚本报"无法加载文件"
```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### Q: `python` 命令找不到
重新安装 Python，勾选 "Add Python to PATH"，然后重启终端。

### Q: 飞书 Bot 发消息没反应
1. 检查 `python cli.py bot status` 是否显示运行中
2. 检查 `logs/feishu_bot.log` 有无报错
3. 检查飞书开放平台的事件订阅 URL 是否正确

### Q: 端口 8765 被占用
```powershell
# 找到占用进程
netstat -ano | findstr "8765"
# 停止占用
taskkill /PID <PID> /F
```
