# DeepRead 手机端使用指南

本指南面向 Trial 体验版用户，默认入口是手机飞书 Bot。

---

## 安装后 3 步开始

1. 安装并配置（见 [快速启动](快速启动-新手版.md)）
2. 启动飞书 Bot：`.\start-bot.ps1`
3. 手机打开飞书，给 Bot 发一条消息

## 常用口令

### 开始阅读

```
你好                          → 看到欢迎菜单
我想开始读一本书                → Bot 帮你建议阅读模式
精读《思考快与慢》第5章         → 直接开始精读
读《被讨厌的勇气》第一夜        → 开始命题辨析模式
```

### 阅读中

```
继续 / 下一阶段                → 进入下一个精读阶段
我不太确定 / 换个说法           → 让 Bot 换个角度追问
跳过                          → 跳过当前阶段
总结一下                      → 让 Bot 给一个总结（会提示"我是教练不是讲解员"）
```

### 模式切换

```
这本书适合什么模式？
切换成考试模式
用工具书模式读
查看当前模式
```

### 其他

```
查看进度
复习
慢思考
帮助
```

## 语音输入技巧

飞书支持语音转文字。长段回答建议用语音：
- 每段回答控制在 30 秒以内
- 先想好再说，不用边想边说
- 说完后可以再发文字补充

## 预期对话长度

- 每篇文章（小节）：BOT 会追问 2-4 轮
- 每章完整精读：大约 10-20 轮对话
- 每次对话建议控制在 15 分钟以内

## 笔记在哪

- **Trial 版**：笔记默认保存在 `deepread/notes/` 目录
- **Personal 版**：笔记保存在 Obsidian vault 的 `读书/笔记/` 目录
- 路径可在 `config.yaml` 中修改
- 服务器版飞书 Bot 默认会在最终笔记生成后，把 `.md` 文件作为附件发回聊天

## 出错怎么办

1. 给 Bot 发 `帮助`
2. 在电脑上打开 Web 控制台 `http://127.0.0.1:8765/`
3. 查看健康检查页面
4. 查看 `logs/feishu_bot.log` 日志文件
5. 运行 `python cli.py doctor` 诊断

## 飞书通知

Bot 消息走飞书通知机制。如果没收到回复：
- 检查 Bot 是否运行：`python cli.py bot status`
- 确认飞书开放平台事件订阅 URL 配置正确
