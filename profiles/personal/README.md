# DeepRead 完整版 (Personal)

## 这是什么

完整功能版：飞书 + Obsidian + LLM-Wiki + 概念卡 + 认知画像。适合长期个人知识建设。

## 配置路径

```powershell
.\install.ps1 -Profile personal
# 或
python init.py --profile personal
```

## 完整能力

- 9 种阅读模式覆盖主要书籍类型
- Obsidian 三段式笔记 + 自动双链
- LLM-Wiki 概念路由和语义增强
- 概念卡完整扫描和别名链接
- 认知画像追踪思维模式
- 飞书 Bot + Web 控制台
